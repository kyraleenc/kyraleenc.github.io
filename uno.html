UNO Game Rules

If you have played the UNO card game before, you might be familiar with the official UNO Game Rules card. This project is a creative reimagination of that, using Adobe InDesign and Illustrator.

❮ ❯

Click here to view the complete Game Rules book.

Challenge

Create a colourful and family-friendly book design for the UNO card game.

User Analysis

The main users/customers of the UNO card game are kids, friends, and family members. These people are trying to understand the game rules quickly. The book should contain the same information and content as the original official UNO Game Rules card but look more colourful and visually appealing.

Result

The result of the project is a multi-paged and colourful booklet with the information spread out. The colours and illustrations help users find the information they need quickly and make the progress more enjoyable.
