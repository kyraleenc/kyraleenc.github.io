MMEX User Guide

Money Manager Ex is an open-source software for personal financial management. I contributed to the project by developing a new user guide to replace their existing one (which was last updated over 10 years ago). I developed this document in both InDesign and Markdown.

Challenge

Develop a comprehensive user guide for MMEX in English, both in Word and InDesign. The User Guide shall be converted to Markdown afterwards.

User Analysis

The main users of MMEX are people (young adults to adults) who are interested in managing their finance.

The book should contain information that would:
  • help them get started with the software
  • progressively disclose more advanced information
  • include basic personal finance information as references

After extensively researching, planning, and interviewing Subject Matter Experts, I organized the topics into 6 categories: Getting Started, Transactions, Investments, Assets and Budgeting, Database and Reports, and Personal Finance Concepts.
